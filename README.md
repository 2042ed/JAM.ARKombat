# ARKombat - card game
ARKombat è un gioco di carte e combattimento a colpi di palle di fuoco o tronchi rotanti. Con solo otto carte in mano, iniziare è facile, smettere un po' meno.  
2 giocatori, dai 5 anni, durata 5/15 minuti.
